# /phase-0-intake — PRD Intake (Blueprint Phase 0)

Execute PROJECT_FRAMEWORK.md Phase 0 on the PRD in this project:

1. Read `PRD.md` end to end.
2. Produce the **PRD Read-Back** — 5 bullets: problem, target user, success criteria, hard constraints, explicit non-goals.
3. Rewrite every success criterion into falsifiable form: `X% of [user] will [action] within [timeframe]`. Anything that can't be made falsifiable → flag as non-goal or research question. Never leave vague.
4. List every ambiguity and unstated assumption. None is too small.
5. Produce the **Prior Art & Steal List** — 3–5 products that solved the same core journey, each with the specific mechanic worth copying.
6. Write all of it into `Documentation/solution/SOLUTION_BRIEF.md` (Read-Back + Prior Art sections).

**Gate:** Stop. Present the read-back. Do not start Phase 1 until Gaurav says "Read-back approved." Implied approval doesn't count.
